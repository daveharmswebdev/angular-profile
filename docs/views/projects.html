Node Cal

A Node.js app that successfully reverse engineers the Bash “cal” command line executable. Success is determined via Mocha/Chai testing. The standard out of this app matches 100% the standard out of the bash cal for month and year. Developed app with TDD principles. Achieved 100% coverage according to the Istanbul coverage tool.
